# 30 Days Java Challenge 
## Day 01 - Even or Odd Number Checker
### Problem Statement
Write a Java program to check whether a given number is Even or Odd.
### ▶ Output
Enter a number: 15  
15 is Odd
Enter a number: 24  
24 is Even

# Day 02 - Prime Number Checker
## Problem Statement
Write a Java program to check whether a given number is a prime number or not.
## Concepts Used
- For Loop
- If-Else Statement
- Modulus Operator (%)
## Sample Output
Enter a number: 7
7 is a Prime Number

# Day 03 - Factorial Program
## Problem Statement
Write a Java program to find the factorial of a given number.
## Example
Input:
5
Output:
120
## Concepts Used
- Scanner Class
- for Loop
- Variables
- Factorial Logic
